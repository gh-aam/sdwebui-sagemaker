# [SD WebUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui) on [SMSL](https://studiolab.sagemaker.aws)

## WebUI
* **Setup**
```
cd /tmp && wget https://github.com/gh-aam/sdwebui-sagemakersl/raw/main/setup.sh && sh setup.sh
```

* **Launch**
```
cd ~/stable-diffusion
```
```
sh webui.sh
```

## Extra
* **Storage**
```
cd ~/stable-diffusion
```
```
sh storage.sh
```
